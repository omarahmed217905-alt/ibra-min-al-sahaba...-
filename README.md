# عبرة من الصحابة 🌙

موقع دعوي تعليمي تفاعلي يقدم قصص مختارة من الصحابة رضي الله عنهم مع الدروس المستفادة منها، وتصميم جميل وسهل الاستخدام.

## ✨ مميزات الموقع

- تصميم داكن أنيق مستوحى من الطابع الإسلامي
- 5 قصص من الصحابة مع نصوص كاملة
- صندوق تفاعلي للدروس المستفادة (Accordion)
- أزرار إعجاب تفاعلية
- شريط تقدم القراءة
- تأثيرات ظهور سلسة عند التمرير
- متوافق مع الجوال (RTL كامل)

## 📖 القصص الموجودة حالياً

1. **حدير رضي الله عنه** - قصة الوفد اليمني وذكر الله
2. **خوات بن جبير رضي الله عنه** - قصة الخجل والتوجيه النبوي الرقيق
3. **هاشم بن عتبة (المِرقال) رضي الله عنه** - شجاعته في معركة القادسية
4. **زيد بن عمرو بن نفيل** (مات قبل البعثة) - رحلة البحث عن التوحيد
5. **عباد بن بشر الأنصاري رضي الله عنه** - الخشوع في الصلاة رغم الإصابة

## 🚀 كيفية تشغيل الموقع محلياً

1. حمل المجلد كاملاً
2. افتح ملف `index.html` في أي متصفح
3. استمتع بالقصص والدروس!

## 🌐 نشر على GitHub Pages

1. ارفع المجلد كاملاً على GitHub
2. ادخل على إعدادات الريبو → Pages
3. اختر الفرع `main` والمجلد `/ (root)`
4. الموقع هيشتغل على: `https://yourusername.github.io/ibra-min-al-sahaba`

## 📹 ملاحظة عن الفيديوهات

تم تحويل كل الفيديوهات إلى **بطاقات روابط يوتيوب** جميلة ومناسبة للنشر على GitHub.

**طريقة إضافة روابطك:**
1. ارفع كل فيديو على يوتيوب (يفضل تكون "غير مدرجة - Unlisted")
2. انسخ رابط المشاركة (Share)
3. افتح ملف `index.html`
4. ابحث عن التعليق `<!-- TODO: ضع رابط يوتيوب هنا -->`
5. غيّر `href="#"` إلى رابط يوتيوب الفعلي

مثال:
```html
<a href="https://youtu.be/YourVideoID" target="_blank" class="watch-btn">
```
